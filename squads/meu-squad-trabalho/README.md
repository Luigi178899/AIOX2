# meu-squad-trabalho

Squad especializado em análise técnica profunda e pesquisa de domínio.

## 🤖 Agentes
- **Specialist**: Analista principal para execução técnica e análise de rigor.
- **Example Agent**: Suporte operacional e validação de fluxo.

## 🚀 Comandos Principais
- `*analyze`: Inicia análise profunda de um tema.
- `*research`: Executa pesquisa técnica externa.
- `*example-task`: Validação de infraestrutura.

## 🤖 IDE Sync
- **Claude Code**: `/Trabalho:agents:specialist`
- **Cursor**: `@specialist`
- **Gemini**: `.gemini/agents/`

## 🛠️ Stack Técnica
- Node.js v20+
- AIOX Core 2.1
